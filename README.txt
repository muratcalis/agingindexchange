## How to install
    - Download and install Pycharm
    - You can automatically download the libraries specified in the scripts
    - First run the program file starting with m1. It will calculate the degrees and degree changes of the genes by mapping them to -1 0 1 values and save them in separate files.
    - After the run is complete, run the file starting with m2. Map the change to 0 -1 and 1 values
    - all results will be written to the output folder.

###Directories
    - mapped contains the network files to be used

------------------------------------------------------------------------------------------------------------

## Nasıl kurulur
    - Pycharm'ı indirin ve kurun
    - Komut dosyalarında belirtilen kütüphanleri otomatik olarak indirebilirsiniz
    - önce m1 ile başlayan program dosyasını çalıştırın. Genlerin derecelerini ve derece değişimlerini -1 0 1 değerlerine map ederek hesaplayıp, ayrı ayrı dosyalara kaydedecektir.
    - Çalışma tamamlandıktan sonra m2 ile başlayan dosyayı çalıştırın. Değişimi 0 -1 ve 1 değerlerine map ederek
    - tüm sonuçlar output klasörüne yazılacaktır.

###Dizinler
    - mapped kullanılacak olan network dosyalarını içerir
